# ezdml_web

Ezdml_web是EZDML的web版本，基于Vue 3 (TypeScript) 和Vite开发。EZDML是一款国产开源免费跨平台的轻量级数据库建模工具，具有轻巧、快速、简便而功能强大的特点，支持AI辅助设计，可轻松地进行数据库表结构设计，建立界面数据模型，支持通过自定义脚本模板生成测试数据、界面原型和前后端代码文件。

在线体验: http://www.ezdml.com/v/

## 安装依赖

克隆代码到本地目录后执行安装：

```sh
npm install
```

### 编译并调试

```sh
npm run dev
```

### 打包发布：

```sh
npm run build
```
